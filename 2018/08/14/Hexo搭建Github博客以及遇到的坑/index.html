olive's Blogs

github, hexo

前言

网上关于hexo+GitHub搭建博客的文章一搜一大把，看了很多，也折腾了会，原因在于同一台电脑配置多个git账户(公司+个人)、部署上传慢等问题。对于遇到的问题记录下来，以便再次遇到时花费更少的时间去解决。

正文

如果没有域名看这篇，如果有域名的话 这篇 很不全。
